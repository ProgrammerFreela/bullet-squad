# Bullet Squad

Projeto Unity 2D run-and-gun para a atividade A4 de Programacao de Jogos.

Requisitos do trabalho:
- jogo 2D ou 3D;
- pelo menos 4 fases;
- menu principal;
- executavel independente;
- GDD com integrantes na capa;
- anexo com snapshots do jogo em execucao e texto explicativo/storyboard.
