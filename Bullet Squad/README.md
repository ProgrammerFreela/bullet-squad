# Bullet Squad

Projeto Unity 2D run-and-gun para a atividade A4 de Programacao de Jogos.

## Estado atual

- Menu principal jogavel.
- Quatro fases em `Assets/Scenes/Level01.unity` ate `Level04.unity`.
- Jogabilidade manual: movimento, pulo, tiro e granada.
- Inimigos com IA simples, pickups, chefes e HUD.
- Assets autorais simples gerados no proprio projeto.
- GDD inicial em `Documentation/GDD_BulletSquad.md`.

## Controles

- `A/D` ou setas: mover.
- `W`, seta para cima ou `Space`: pular.
- `J`, `Ctrl` esquerdo ou mouse esquerdo: atirar.
- `K` ou mouse direito: granada.
- No menu: `Enter` inicia, `1-4` abre uma fase.

Requisitos do trabalho:
- jogo 2D ou 3D;
- pelo menos 4 fases;
- menu principal;
- executavel independente;
- GDD com integrantes na capa;
- anexo com snapshots do jogo em execucao e texto explicativo/storyboard.
